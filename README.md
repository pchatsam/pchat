# pchat
A pure frontend P2P messaging library built on PeerJS with end-to-end encryption.

## Features

- **Peer-to-peer messaging** — data flows directly between users, no server relay
- **End-to-end encryption** — RSA 2048 key exchange with AES-GCM message encryption
- **Rich media** — text, images, files, and voice messages
- **Voice calls** — real-time WebRTC audio
- **Group chats** — client-side contact grouping
- **Offline storage** — all data encrypted in browser IndexedDB
- **Serverless** — only requires a signaling server (PeerJS), no custom backend needed
